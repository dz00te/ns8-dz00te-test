# dz00te modules index for NS8

This is the dz00te forge NS8 index of modules.

Metadata are built every 4 hours at 00:25, 06:25 ,12:25, 18:25 UTC and on each commit to the `main` branch.

To use this as an NS8 repository, set the following URL:

    https://forge.dzlab.net/ns8/updates/

Alternative URL:

    https://raw.githubusercontent.com/dz00te/ns8-dz00te/repomd/ns8/updates/
